# UK_energy_mix Data Project in Power BI
Data Analysis of the UK power generation by fuel type in Power BI. Data has been downloaded from https://www.neso.energy/data-portal/historic-generation-mix/historic_gb_generation_mix . This large dataset contains half-hourly time-stamped data for UK's power generation fuel-mix for the last 16 years.
